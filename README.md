# Expense-Tracker
A simple and responsive Expense Tracker web app built using HTML, CSS, and JavaScript. It helps users manage their income and expenses while tracking their balance in real-time.

Live Demo - 

🚀 Features
📊 Track income and expenses
💵 Real-time balance calculation
➕ Add transactions (income/expense)
❌ Delete transactions
📋 Transaction history
🎨 Clean and modern UI

🛠️ Tech Stack
HTML5
CSS3
JavaScript

⚙️ How It Works
Enter a description and amount
Use:
Positive (+) amount → Income
Negative (-) amount → Expense
Click Add Transaction
Balance, income, and expenses update automatically
